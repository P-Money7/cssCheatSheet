Position

Value Declaration/Example Code Description
Static position: static; Positioning is the default way elements are positioned on the page
Relative position: relative; Moves the element, but still consumes some space in the original position
Absolute position: absolute; Takes elements completely out of the normal document flow to sit in a separate layer
Fixed position: fixed; The element is removed from the normal document flow, and no space is created for the element in the page layout.
Sticky position: sticky; Takes element out of normal flow and "sticks" to the screen when scrolling

Color

Value Declaration/Example Code Description
Text Color color: coral; changes text to named color
Background Color background-color: aqua; fills background with named color
Opacity opacity: 0.75; adjusts transparency level of element thus blending colors
Background Image background-image: url(../file.jpg); fills background of an element with an image
